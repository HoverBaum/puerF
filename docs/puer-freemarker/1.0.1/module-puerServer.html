puerServer

puerServer module for puerFreemarker.

This module provides a single function to start a puer Server.

Source:
  • puerServer.js, line 1

(require("puerServer"))(routesFile, options, callback)

Source:
  • puerServer.js, line 48

Start a puer server to serve files and watch for changes.

Parameters:
Name Type Description
routesFile array

All files containing info about mocked routes. Paths need to be relative to current working directory.
options object

Options to run the server with.

Properties
Name Type Attributes Default Description
templatesPath string <optional>

Root folder for FTL template files.
dir string <optional>

Root folder for static files to serve.
port number <optional>
 8080

The port to use for the server.
watch string <optional>
 'js css html xhtml ftl'

Filetypes (seperate them by pipes) to watch for changes.
ignored regEx <optional>
 /node_modules/

Files to exclude from watching.
localhost boolean <optional>
 false

Use localhost instead of 127.0.0.1.
browser boolean <optional>

Automatically open a browser for the user.
callback function

Function to call once done.
